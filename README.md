🖥️ MiniShell

A simple Unix-like command-line shell implemented in C that can execute basic system commands.

📌 Overview

MiniShell is a lightweight shell program that mimics basic functionality of a Linux terminal.
It takes user input, processes commands, and executes them using system calls.

🚀 Features
Execute basic commands (ls, pwd, echo, etc.)
Supports command-line input
Uses fork() and exec() system calls
Simple and easy to understand
